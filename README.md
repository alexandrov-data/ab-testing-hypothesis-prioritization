# Приоритизация гипотез и анализ A/B-теста

## О проекте
В проекте проводится приоритизация продуктовых гипотез и анализ результатов A/B-теста интернет-магазина.

Цель — определить, какие гипотезы стоит запускать в первую очередь, и на основе результатов эксперимента оценить, влияет ли изменение на ключевые бизнес-метрики.

## Бизнес-задача
Интернет-магазину важно принимать решения о развитии продукта на основе данных, а не предположений.

Анализ помогает ответить на ключевые бизнес-вопросы:
- какие гипотезы имеют наибольший потенциал для роста бизнес-метрик;
- есть ли различия между группами A/B-теста;
- стоит ли внедрять изменение, продолжать тест или остановить его.

## Задачи
- подготовить и проверить данные для анализа;
- приоритизировать гипотезы с помощью фреймворков ICE и RICE;
- рассчитать и визуализировать кумулятивные метрики A/B-теста;
- сравнить группы по выручке, среднему чеку и конверсии;
- выявить аномальные значения в данных;
- оценить влияние выбросов на результаты теста;
- проверить статистическую значимость различий между группами;
- сформулировать итоговое решение по результатам эксперимента.
